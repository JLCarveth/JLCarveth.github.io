Elements of HTML5

    The Heading Element

    The heading element is used to title a section of content. There are 5 sizes of heading, with h1 being the largest, and h5 being the smallest. An example of HTML5 heading elements can be seen below.

      Syntax: 
                          <h1>Largest Title</h1>
                    <h2>Large Title</h2>
                    <h3>Medium Title</h3>
                    <h4>Small Title</h4>
                    <h5>Smalest Title</h5>
      Example:

      Largest Title

      Large Title

      Medium Title

      Small Title

      Smalest Title

    The Line-Break Element

    This element is used to add a line-break to an HTML page, causing the next element to be displayed opn the next line. To create a line break, use the <br> tag.

    The Blockquote Element

    Used to give emphasis to a block of text by indenting the paragraph. A blockquote is created with the <blockquote> </blockquote> tag.

    Lists

    Unordered Lists
    • Unordered lists display list items with bullet points. An unordered list is declared with the <ul> tag, and elements are added as children using the <li> tag.
    • This list is an example of an unordered list.
    Ordered Lists
    1. Ordered lists are declared with the <ol> tag.
    2. In an ordered list, list items are denoted numerically.
    Description List
    • A Description List is used to display a list of terms.
    • <dl> declares a Description List
    • <dt> declares a term/phrase
    • <dd> declares a description
    • Example:
      Sun
      A hot ball of gas.
      Cat
      A cute animal.
