Node.js / Express.js Blog Backend System

This was my first real project with Node.js and it was quite fun to program. This is an HTTP api backend built for a blogging website. It includes several prominent features:

  • Role-Based access control: Any HTTP route can easily be set to require a valid token issued to a user with the correct permissions. Roles can be set within the config file, and each role can have many permissions.
  • User Authentication: Goes hand-in-hand with RBAC. Users can register, login, and recieve tokens from the system. Tokens must then be supplied with future requests to access sensitive HTTP routes.
  • EJS Support: This backend service can also serve static html files to clients using EJS templates.

See this project on GitHub.
